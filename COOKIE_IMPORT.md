# 🍪 Инструкция по импорту кукис

## Зачем это нужно

Вместо ручного логина в каждый сервис, приложение скопирует сессии из твоего браузера (Chrome или Edge).

---

## 📋 Пошаговая инструкция

### Шаг 1: Залогинься в браузере

Открой в своём **обычном браузере** (Chrome или Edge) и залогинься:

1. https://chat.openai.com → Залогинься в ChatGPT
2. https://claude.ai → Залогинься в Claude
3. https://gemini.google.com → Залогинься в Gemini
4. https://x.com → Залогинься в X (для Grok)

**Важно:** Оставайся залогиненным, не выходи.

---

### Шаг 2: Закрой браузер

**Полностью закрой** Chrome или Edge (все окна).

Это нужно, чтобы освободить доступ к базе данных кукис.

---

### Шаг 3: Переустанови зависимости

Открой PowerShell в папке проекта:

```powershell
npm install
```

Это установит `better-sqlite3` для чтения кукис.

**Важно:** Если выдаст ошибку компиляции — не страшно, попробуй запустить приложение.

---

### Шаг 4: Запусти приложение

```powershell
npm start
```

---

### Шаг 5: Импортируй куки

В приложении:

**Вариант А: Через меню**
1. Нажми `Tools` → `Import Cookies from Browser`
2. Или горячая клавиша: `Ctrl+I`

**Вариант Б: Через консоль (если меню не работает)**
1. Нажми F12 (откроется DevTools)
2. В консоли напиши:
```javascript
require('electron').ipcRenderer.send('import-cookies')
```

---

### Шаг 6: Проверь результат

В консоли (F12) должно появиться:

```
🔍 Searching for browser cookies...
✓ Found Chrome cookies
✓ Found 15 cookies for chat.openai.com
✓ Found 8 cookies for claude.ai
...
✅ Cookie import completed!
```

После этого все WebView перезагрузятся и ты будешь автоматически залогинен!

---

## ⚠️ Возможные проблемы

### Ошибка: "Failed to copy cookie database"

**Причина:** Браузер ещё запущен.

**Решение:** 
1. Полностью закрой Chrome/Edge (через Диспетчер задач убей процесс)
2. Попробуй снова

---

### Ошибка: "better-sqlite3 не найден"

**Причина:** Не установилась зависимость.

**Решение:**
```powershell
npm install better-sqlite3 --save
```

Если не помогло — попробуй установить build tools:
```powershell
npm install --global windows-build-tools
```

---

### Ошибка: "No supported browser found"

**Причина:** Используешь Firefox/Brave/другой браузер.

**Решение:** 
Пока поддерживается только Chrome и Edge. Используй один из них для логина.

---

## 🔄 Повторный импорт

Если вышла сессия:

1. Залогинься заново в браузере
2. Закрой браузер
3. В приложении: `Ctrl+I` (Import Cookies)
4. Готово

---

## 🎯 После успешного импорта

Попробуй отправить тестовое сообщение во все чаты:

```
Привет! Это тест из агрегатора
```

Должно отправиться во все активные панели.
