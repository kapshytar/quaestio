🚀 Quaestio v0.1

Работай с 4 AI одновременно

✨ Что это?
Electron-приложение для одновременной работы с ChatGPT, Claude, Gemini и Grok

📦 Установка и запуск

1 Установите Node.js (если ещё нет)
Скачайте с nodejs.org (версия 18 или выше)
2 Запустите приложение
Windows: Дважды кликните start.bat
Linux/Mac: Выполните ./start.sh

Или вручную:
npm install
npm start
3 Импортируйте cookies
Нажмите кнопку "Import 🍪" и выберите файл cookies.json
(Экспортируйте cookies из расширения Cookie-Editor)

✨ Возможности

✅ Отправка во все чаты
✅ Импорт cookies
✅ Редактируемые URL
✅ Управление масштабом
✅ Навигация (back/forward)
✅ Компактный UI

⌨️ Горячие клавиши

  • Enter — отправить сообщение
  • Shift+Enter — новая строка
  • Ctrl+I — импорт cookies
  • Ctrl+R — перезагрузить все окна
  • F12 — открыть DevTools
  • Ctrl+Plus/Minus — изменить масштаб
  • Ctrl+0 — сбросить масштаб

🍪 Как получить cookies?

1 Установите расширение Cookie-Editor для вашего браузера
2 Зайдите на нужный AI-сервис и авторизуйтесь
3 Откройте Cookie-Editor → Export → JSON
4 Сохраните файл и импортируйте в Quaestio

⚠️ Известные проблемы

Gemini зацикливается
На gemini.google.com может быть бесконечный редирект.
Используется aistudio.google.com вместо этого.
Селекторы могут устареть
Если AI-сервисы обновят интерфейс, может потребоваться обновление селекторов в коде.

📁 Файлы проекта

  • main.js — главный процесс Electron
  • index.html — интерфейс приложения
  • renderer.js — логика фронтенда
  • preload.js — безопасный мост IPC
  • cookie-import-simple.js — импорт cookies
  • package.json — конфигурация проекта
