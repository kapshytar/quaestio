# 🚀 БЫСТРЫЙ СТАРТ

## Windows

1. Дважды кликните `start.bat`
2. Дождитесь установки зависимостей (только первый раз)
3. Приложение запустится автоматически

## Linux / macOS

```bash
./start.sh
```

Или вручную:

```bash
npm install
npm start
```

## Первая настройка

1. **Импортируйте cookies** для авторизации:
   - Нажмите кнопку "Import 🍪" в правом верхнем углу
   - Выберите файл cookies.json (экспортированный из Cookie-Editor)
   - Все окна перезагрузятся автоматически

2. **Проверьте авторизацию**:
   - ChatGPT: должен показать ваш аккаунт
   - Claude: должен открыться чат
   - Gemini: должен быть доступен AI Studio
   - Grok: должен загрузиться интерфейс X.com

3. **Отправьте тестовое сообщение**:
   - Введите "Hello!" в поле ввода внизу
   - Нажмите Enter или кнопку "Send"
   - Сообщение отправится во все активные чаты (отмеченные галочками)

## Что делать если не работает?

### "npm: command not found"
→ Установите Node.js с https://nodejs.org/

### Не загружается один из AI-сервисов
→ Проверьте cookies для этого сервиса

### Сообщение не отправляется
→ Откройте DevTools (F12) и проверьте консоль на ошибки

### Gemini зацикливается
→ Это известная проблема с gemini.google.com, используется aistudio.google.com

## Горячие клавиши

- `Enter` - отправить сообщение
- `Shift+Enter` - новая строка
- `Ctrl+I` - импорт cookies
- `Ctrl+R` - перезагрузить все окна
- `F12` - DevTools
