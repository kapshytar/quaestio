# 🚀 Быстрый старт (Quickstart)

## 1. Подготовка
Убедитесь, что у вас установлен **Node.js** (версия 18+).

### macOS (дополнительно)
- Установите Xcode Command Line Tools:
```bash
xcode-select --install
```
- Если `start.sh` не запускается, дайте права:
```bash
chmod +x start.sh
```

## 2. Установка
Склонируйте проект и выполните:
```bash
npm install
```

## 3. Запуск
В Windows используйте `start.bat` или команду:
```bash
npm start
```

В macOS/Linux можно использовать:
```bash
./start.sh
```

## 4. Авторизация (Cookies)
Самый быстрый способ войти во все аккаунты — импорт куки из браузера.
1. Залогиньтесь в Chrome или Edge в ChatGPT, Claude, Gemini, X.com.
2. Закройте браузер полностью.
3. В приложении нажмите `Ctrl+I` (Import Cookies).
4. Подробности см. в [docs/COOKIE_IMPORT.md](docs/COOKIE_IMPORT.md).

## 5. Использование
- **Сетка из 4 слотов:** По умолчанию открыты ChatGPT, Claude, Gemini, Grok.
- **Выбор слотов:** В верхней панели можно отключить слоты галочками (✓).
- **Отправка:** Введите текст в нижнем поле и нажмите **Send** (или `Enter`).
- **Слияние (Merge):** Нажмите **Merge ▸**, введите API ключ вашего провайдера и нажмите **Run Merge** для синтеза всех ответов.

## 6. Если что-то не работает
- **Gemini:** Используется Google AI Studio для стабильности.
- **Grok:** Требуется активная подписка X Premium.
- **DevTools:** Нажмите `F12` для просмотра логов консоли.

См. также [docs/FIXES.md](docs/FIXES.md) для решения частых проблем.
