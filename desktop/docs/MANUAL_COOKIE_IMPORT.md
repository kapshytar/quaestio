# 🍪 Инструкция: Ручной импорт кукис

## 📋 Пошаговая инструкция

### Шаг 1: Установи расширение для экспорта кукис

**Для Chrome/Edge:**

1. Открой Chrome Web Store: https://chrome.google.com/webstore
2. Найди расширение **"EditThisCookie"** или **"Cookie-Editor"**
3. Установи расширение

**Рекомендую:** Cookie-Editor (простой и надёжный)

---

### Шаг 2: Экспортируй куки для каждого сайта

Для **каждого** из этих сайтов:

#### 2.1 ChatGPT

1. Открой https://chat.openai.com
2. Залогинься (если ещё не)
3. Кликни на иконку расширения Cookie-Editor
4. Нажми кнопку **Export** (справа вверху)
5. Скопируй JSON в буфер обмена
6. Создай файл `chatgpt-cookies.json` и вставь туда

#### 2.2 Claude

1. Открой https://claude.ai
2. Залогинься
3. Кликни на иконку Cookie-Editor
4. Export → скопируй
5. Создай файл `claude-cookies.json`

#### 2.3 Gemini

1. Открой https://gemini.google.com
2. Залогинься
3. Cookie-Editor → Export
4. Создай файл `gemini-cookies.json`

#### 2.4 Grok (опционально)

1. Открой https://x.com
2. Залогинься
3. Cookie-Editor → Export
4. Создай файл `grok-cookies.json`

---

### Шаг 3: Объедини все куки в один файл (опционально)

Можешь либо импортировать файлы по отдельности, либо объединить их.

**Для объединения:**

Создай файл `all-cookies.json`:

```json
[
  ... куки из chatgpt-cookies.json ...,
  ... куки из claude-cookies.json ...,
  ... куки из gemini-cookies.json ...,
  ... куки из grok-cookies.json ...
]
```

Просто скопируй содержимое всех файлов в один массив.

---

### Шаг 4: Импортируй куки в приложение

1. Запусти приложение:
   ```powershell
   npm start
   ```

2. В меню нажми:
   **Tools → Import Cookies from File...**
   
   Или горячая клавиша: **Ctrl+I**

3. Выбери файл с куками (например `all-cookies.json`)

4. Должно появиться сообщение:
   ```
   Cookies imported! WebViews reloaded.
   ```

---

## 📊 Что увидишь в консоли

```
📥 Importing cookies from: C:\Users\...\all-cookies.json
✅ Import complete: 45 imported, 0 errors
```

После этого все панели перезагрузятся и ты будешь залогинен!

---

## 🔍 Альтернативный метод (если расширение не работает)

### Через DevTools браузера

1. Открой сайт (например https://chat.openai.com)
2. Нажми **F12** (DevTools)
3. Вкладка **Application** → **Cookies**
4. Выбери домен (например `chat.openai.com`)
5. Правый клик на кукис → **Copy all as JSON** (если доступно)

Если нет кнопки "Copy as JSON", используй консоль:

```javascript
copy(document.cookie)
```

Это скопирует куки в формате строки. Но лучше используй расширение.

---

## ⚠️ Важно

- **Не делись файлами с куками** — это твои сессии!
- Храни их локально
- После импорта можешь удалить JSON файлы

---

## 🎯 После импорта

Попробуй отправить тестовое сообщение:

```
Привет! Тест из агрегатора
```

Должно отправиться во все залогиненные панели.

---

## ❓ Проблемы?

**Не импортируются куки:**
- Проверь формат JSON (должен быть массив объектов)
- Убедись, что в куках есть поля `name`, `value`, `domain`

**Не залогинивает:**
- Попробуй экспортировать куки заново
- Убедись, что в браузере ты действительно залогинен

**Куки быстро протухают:**
- Некоторые сессии действуют 1-7 дней
- Придётся переимпортировать
